# 毕设前端代码
套用vue-admin-template框架

幼儿园共建共享绘本图书馆管理平台的设计与实现


http://localhost:9529/


## Build Setup


```bash
# clone the project
git clone https://github.com/PanJiaChen/vue-admin-template.git

# enter the project directory
cd vue-admin-template

# install dependency
npm install

# develop
npm run dev
```
